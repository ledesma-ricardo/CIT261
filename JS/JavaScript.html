Conditional Statements

If..else Statement Example:

This example use a function to prompt the user to specify his/her age and run an if/else code so, if the user's age is greater than or equal to 18 years, he will let the user know that he is an adult, otherwise he will return a message stating that the user is not an adult. Below is the code used.

         <script type="text/javascript">
            function age() {
              var age = prompt("Please enter your age?");
		      
              if (age >= 18) {
		      alert("You are an adult");} 
              else{
                    alert("You are NOT an adult");}
               }
         </script>

Loops

For Loop Example:

In this example I used a for loop statement to show all my classmates names that are store in array. Below is the code used.

             <script type="text/javascript">
                function ClassmatesForLoop() {
                var names = ["Elias", "Marcelo", "Vanessa", "Jean", "Alfredo", "Carlos"];
                var classMate = "";
                for (var i = 0; i < names.length; i++) {
                    classMate += names[i] + "<br>";
                }
                    document.getElementById("MatesNames").innerHTML = classMate;
                }
             </script>

While Loop Example:

In this example I used a While loop statement to show all my classmates names that are store in array. Below is the code used.

             <script type="text/javascript">
               function ClassmatesWhileLoop() {
                var names = ["Elias", "Marcelo", "Vanessa", "Jean", "Alfredo", "Carlos"];
                var i = 0;
                var classMate = "";
                while (names[i]){
                    classMate += names[i] + "<br>"; i++;
                }
                    document.getElementById("MatesNames_while").innerHTML = classMate;
                }
             </script>
